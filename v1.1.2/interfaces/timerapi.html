on

  • on<T>(eventType: T, cb: TimerListener<T>): () => void
    • Defined in types.ts:89

    Listen to specific events. Similar to subscribe, but only subscribes to events of given type. The given callback will be called when the eventType event is emitted. The callback receives the same arguments as subscribe's callback, but the second argument EventType will always be of the type given here. Returns a function that can be called to unsubscribe this listener.

    Type parameters

    • T: "play" | "pause" | "setTime" | "reset" | "update" | "finish" | "setDuration" | "setUpdateInterval"

subscribe

  • subscribe(cb: TimerSubscriber): () => void
    • Defined in types.ts:79

    Subscribe to time updates. The given callback function will be called anytime the state changes. The callback gets two arguments:

    1. The Timer context, including state and API functions,
    2. A TimerEvent object, that contains a type property, informing what caused the update. Returns a function that can be called to unsubscribe.
